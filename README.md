### Screenshot
![light theme](https://github.com/foxching/weather/blob/master/public/images/screenshot/weather2.png)


### Installation

Install all the dependencies

```sh
$ yarn install
```

### Serve
To serve in the browser  

```sh
$ yarn start
```
Serve on localhost:300

### Build
Compile and build

```sh
$ yarn build
```
